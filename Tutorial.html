Berikut 3 Virus Yang Saya Telah Rangkum

1. Force Shutdown Virus

Deskripsi :

Simpan file tersebut dengan nama : Shutdown.bat

Kemudian ubah tipenya menjadi all files! lalu simpan di bagian desktop Setelah itu buat shortcut yang sama persis dengan aplikasi favourite korban!
Dan, saat korban mengklik file tersebut. File akan aktif dan Duarrrrrr...!!! Komputer korban akan mati secara paksa! (Gunakan Trik Ini Secara Bijak!)


2. Network Virus

Deskripsi :

Simpan file tersebut dengan nama : Network.bat

Kemudian ubah tipenya menjadi all files! sama seperti tadi. lalu simpan Dibagian desktop. Setelah itu buat lagi shortcutnya sama persis pentunjuk diatas!
Saat korban mengklik file tersebut, akses internetnya akan down! Dan korban tidak dapat melakukan browsing!
Cara pemuliahannya ialah : Buka CMD lalu Ketikkan ipconfig/renew


3. Hank Virus

Deskripsi :

Simpan file tersebut dengan nama : Hank.bat

Kemudian ubah tipenya menjadi all files Dan, ikuti langkah seperti yang tadi! Virus ini dapat membuka aplikasi cmd secara
terus-menerus dan membuat komputer menjadi hank!
Jadi, jangan pernah bermain-main dengan virus ini! (Ingat boss gunakan wawasan ini dengan bijak!)
